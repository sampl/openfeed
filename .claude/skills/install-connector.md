# Install an External Connector

This skill helps you install a third-party OpenFeed connector published to npm. External connectors use the `openfeed-connector-` naming prefix and are loaded automatically when installed alongside OpenFeed.

## Instructions

Read the documentation at `docs/connectors/index.md` for full details on how to find, install, and configure external connectors from npm.
