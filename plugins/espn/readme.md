# ESPN plugin

Fetches sports news from ESPN via RSS.

## Supported URL

`https://www.espn.com`

## Options

- `feed` (string) — override the RSS feed URL. Defaults to `https://www.espn.com/espn/rss/news`. Use this to subscribe to sport-specific feeds, e.g. `https://www.espn.com/espn/rss/nfl/news` for NFL or `https://www.espn.com/espn/rss/nba/news` for NBA.

## Render output

`richText` — HTML and plain-text article summary from the RSS feed.
