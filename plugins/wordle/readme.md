# Wordle plugin

Wordle does not have a public API or RSS feed, so this plugin cannot fetch game results automatically.

## Alternatives

**NYT Games via the nyt-crossword plugin** — If you want NYT Games content in your feed, the `nyt-crossword` plugin supports daily crossword puzzles using the NYT API. Configure it with your NYT API key.

**Manual tracking** — You can copy your daily Wordle result and add it to a personal feed or notes app manually.
