# The New Yorker plugin

Fetches articles from The New Yorker via RSS.

## Supported URL

`https://www.newyorker.com`

## Options

- `feed` (string) — override the RSS feed URL. Defaults to `https://www.newyorker.com/feed/everything`.

## Render output

`richText` — HTML and plain-text article summary from the RSS feed.
