# Wall Street Journal plugin

Fetches news from The Wall Street Journal via RSS.

## Supported URL

`https://www.wsj.com`

## Options

- `feed` (string) — override the RSS feed URL. Defaults to `https://feeds.a.dj.com/rss/RSSWorldNews.xml`.

## Render output

`richText` — HTML and plain-text article summary from the RSS feed.

## Note

WSJ requires a subscription for full article content; RSS provides headlines and summaries only.
