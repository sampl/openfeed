# Al Jazeera plugin

Fetches news from Al Jazeera via RSS.

## Supported URL

`https://www.aljazeera.com`

## Options

- `feed` (string) — override the RSS feed URL. Defaults to `https://www.aljazeera.com/xml/rss/all.xml`.

## Render output

`richText` — HTML and plain-text article summary from the RSS feed.
