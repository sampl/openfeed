# CENTCOM plugin

Fetches press releases and news from US Central Command via RSS.

## Supported URL

`https://www.centcom.mil`

## Options

- `feed` (string) — override the RSS feed URL. Defaults to `https://www.centcom.mil/DesktopModules/ArticleCS/Feed.aspx?ContentType=1&Site=575&isdashboardselected=0&max=12`.

## Render output

`richText` — HTML and plain-text article summary from the RSS feed.
